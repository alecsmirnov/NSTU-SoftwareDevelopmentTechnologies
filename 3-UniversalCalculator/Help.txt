Универсальный калькулятор представляет из себя набор из 3-х различных калькуляторов: калькулятор P-ичных чисел, калькулятор простых дробей, калькулятор комплексных чисел. 

Калькулятор P-ичных чисел позволяет проводить стандартные операции над числами в разных системах счиления. Основание системы счисления не может быть меньше 2 и больше 16. Для ввода дробных чисел используется символ "." (точка). 
Длина дробной части при преобразовании составляет 15 символов, для предотвращения ситуаций перевода чисел с бесконечной дробной частью.

Калькулятор простых дробей позволяет проводить стандарнтые операции над простыми дробями. Для ввода знаменателя чисел используется символ "/" (наклонная черта). 

Калькулятор комплексных чисел позволяет проводить стандарнтые операции над комплексными числами. Для ввода мнимой части чисел используется символ "+i". 

Длина чисел не должна превышать 2^64 символа.
